A CMR stands for "Cheetah Mobile Resource".
Many would mistake it for being a custom file type but that would be wrong.
A CMR is a ZLIB image that is compressed and upside-down.

The process goes like this:
Regular image file (ex. PNG, JPG, etc.) gets flipped upside down.
That image is then converted into an RGBA file.
That RGBA file is then compressed and turned into a ZLIB file.
That ZLIB file is renamed to ".cmr".
The file is ready to be used in-game.
